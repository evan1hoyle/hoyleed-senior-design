# Poker Tracker


### Steps 

- requirements
```
pip install ultralytics==8.0.0
```

- Start tracking 
```

python flop.py
options:
  -h, --help      show this help message and exit
  -v, --verbose   Enable verbose output
  -z, --setzones  Sets zones

set zones for first run.

```
- website

```
python -m http.server 8080
navigate to localhost:8080

```
